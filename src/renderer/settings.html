Settings
Port number. This number is used for communication between Slajdomat and the Figma plugin. The default is 3001. If you use a non-default number (because 3001 is used by some other program on your computer) then make sure that the Figma plugin is configured to use the same number.
Paths to ffmpeg and ffprobe. These programs are needed for recording sound. If the two fields below are empty, then Slajdomat could not find these paths, which likely means that you have not installed the programs.
