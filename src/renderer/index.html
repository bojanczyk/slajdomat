Step 1. Choose a folder. Select a folder for your presentations, using ‘Choose presentation folder’  in the ‘File’  menu of this app. This will be the folder where you keep all of your Slajdomat presentations. It can have subfolders. If you copy the presentation folder to your website, it will be viewable online as a web page.

Step 2. Set up Figma. In order to use Slajdomat, you need to have Figma, and the Slajdomat plugin for Figma (the plugin is not the same as this app). To add the Slajdomat plugin to Figma, use ‘Manage plugins...’  in the ‘Plugins’  menu of Figma.

Step 3. Make the presentation in Figma. Create a new ‘Design file’  in Figma, and use the Slajdomat plugin and your amazing drawing skills to create slides.

Step 4. Export from Figma to this app. When you want to run your slides, prese the ‘Export’  button in the Slajdomat plugin in Figma. This will send the slides from Figma to the presentation folder, and you will be able to view them in this app.

There are no presentations in this folder. Create a new presentation using the Slajdomat plugin for Figma.
